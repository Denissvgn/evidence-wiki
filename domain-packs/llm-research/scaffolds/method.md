# Method Page Scaffold

Use this scaffold for reusable LLM research techniques, orchestration patterns, state and memory strategies, evaluation workflows, search procedures, training loops, and tool-use methods. Keep source-specific summaries in `wiki/sources/` and concrete implementations in `wiki/systems/`.

```yaml
---
type: method
created: YYYY-MM-DD
updated: YYYY-MM-DD
source_ids:
  - paper:source-id
summary: One-sentence definition of the method.
method_name: Method name
method_type: orchestration_model | state_memory_strategy | tool_use | evaluation_workflow | search_procedure | training_loop | other
---
```
