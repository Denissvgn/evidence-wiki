# Paper Source Note Scaffold

Use this scaffold for academic papers and technical reports after a normalized source record exists. Keep the note factual and source-grounded. Mark missing evidence as a gap instead of inferring it.

```yaml
---
type: source
created: YYYY-MM-DD
updated: YYYY-MM-DD
source_ids:
  - paper:source-id
summary: One-sentence summary of the paper's contribution.
title: Paper title
authors:
  - Author Name
venue: Venue or source
date: YYYY-MM-DD
doi:
arxiv_id:
url:
normalized_record: sources/normalized/example.md
raw_paths:
  - raw/path
---
```
